## 🔰初めての方

---

### 各種設定

#### セットアップ

```shell
sh ./scripts/setup.sh
```

#### .envファイル設定

ファイルを作成してください。

```shell
touch .env

echo "RUN_PORT = 3000" >> .env
```

```env
RUN_PORT = 3000
```

**[こちらへアクセス](http://localhost:3000)**
